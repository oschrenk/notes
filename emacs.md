# Emacs #

## Installation ##

On Lion

	$ brew install emacs --cocoa --use-git-head --HEAD
	$ cp -r /usr/local/Cellar/emacs/HEAD/Emacs.app /Applications/

## Configuration ##

	$ git clone git://github.com/bbatsov/emacs-prelude.git path/to/local/repo
	$ ln -s path/to/local/repo ~/.emacs.d
